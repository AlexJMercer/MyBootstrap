Bootstrap

O que é?

Bootstrap é uma ferramenta gratuita para desenvolvimento de HTML, CSS e JS. É possível criar rapidamente prototipos de suas ideias ou desenvolva aplicações completas com as variaveis e mixins SASS, sistema de grid responsivo, componentes pré-construidos extensos e plugins desenvolvidos em jQuery.

Início rápido

É possivel fazer o download dos arquivos CSS e JS compilados nesse link, ou utilizar os links abaixo em seu código.

Vantagens

  • Excelente sistema de grid e facilidade para o desenvolvimento de layouts responsivos.
  • Possui uma documentação bem detalhada.
  • Compatibilidade com os diferentes navegadores.
  • Um grande número de componentes desenvolvidos e personalizáveis.
  • O uso de SASS(SCSS) permite a padronização de valores, e também a organização dos componentes e utilidades em diferentes arquivos, para a compilação de todas as classes desenvolvidas em um único arquivo. Abaixo um exemplo simples:
